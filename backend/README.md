# Python Backend

This backend handles:
- Excel file processing
- Column type detection
- Template conversion
- API integrations
- Output generation

## Setup

```bash
# Install Python dependencies
pip install -r requirements.txt

# Create .env file
PORT=5000
DEBUG=True

# Run the server
python app.py
```
